📖 概述

PortraitPay API 允许 AI 平台、创作者工具和任何需要查询肖像版权的应用：

  • 查询特定肖像的授权状态和可用性
  • 通过指纹搜索相似已注册肖像
  • 获取授权价格和使用限制
  • 发起授权申请或购买

Local-First 模式：PortraitPay 采用 Local-First 架构。原始肖像图像保留在用户设备上，只上传不可逆指纹哈希进行检索。您的应用可以在不传输原始图像的情况下进行版权查询。

Base URL:

https://portraitpayai.com/api

响应格式: 所有响应均为 JSON

🔐 认证

API 使用 API Key 进行认证。在请求头中添加：

X-API-Key: your_api_key_here

获取 API Key：

  • 注册 PortraitPay 账户
  • 登录后访问仪表盘获取 API Key
  • 企业客户可申请更高调用额度

📡 API 端点

POST /portrait/search-by-fingerprint

通过指纹哈希搜索已注册肖像（Local-First 模式）

参数 类型 说明
fingerprint_hash string 图像的 dHash/ahash 指纹（16-64字符）
fingerprint_type string "dhash" | "ahash" | "phash"（默认 "dhash"）
threshold float 相似度阈值 0-1（默认 0.35）
limit int 最大返回结果数（默认 5，最大 20）
// Request { "fingerprint_hash": "7f3d8a9c1b4e5d2f6a8b3c5d7e9f1a2", "fingerprint_type": "dhash", "threshold": 0.35 }

响应示例

// 200 OK { "matches": [ { "face_id": 12345, "name": "刘德华", "similarity": 0.92, "authorization_status": "available", "price": 500, "copyright_info": "禁止用于政治目的" } ], "total": 1, "search_id": "audit_12345" }
POST /portrait/authorization-check

查询特定肖像是否可用于特定场景

参数 类型 说明
face_id int 肖像 ID（必填）
usage_type string "ai_training" | "ai_generation" | "advertising" | "entertainment"
commercial_use boolean 是否商业用途
exclusive boolean 是否需要独家授权
// Request { "face_id": 12345, "usage_type": "ai_generation", "commercial_use": true }

响应示例

// 200 OK { "can_use": true, "authorization_status": "available", "price": 500, "license_type": "commercial", "restrictions": ["no_politics", "no_porn"], "requires_contract": false }

授权状态说明

状态 说明
available 可直接使用，无需额外审批
pending_review 正在审核中，暂不可用
requires_manual_approval 需要权利人手动审批
unavailable 不可用（已禁用或已删除）

🖥️ Local-First 实现

PortraitPay 支持 Local-First 模式，保护用户隐私同时实现版权检索。

在浏览器中生成指纹

// 使用 PortraitFingerprint JS 库（见下方链接） const fingerprint = await PortraitFingerprint.generate(imageFile); // 返回 { hash: "7f3d8a9c...", type: "dhash", bits: 64 }

Python 示例

# 安装: pip install imagehash pillow import imagehash from PIL import Image img = Image.open('portrait.jpg') fingerprint = imagehash.dhash(img) # 返回类似 "7f3d8a9c1b4e5d2f"

注意：指纹是单向哈希，无法从指纹还原原始图像。这确保了用户肖像隐私，同时允许有效的相似度匹配。

💻 完整示例

cURL

# 搜索肖像 curl -X POST https://portraitpayai.com/api/portrait/search-by-fingerprint \ -H "Content-Type: application/json" \ -H "X-API-Key: your_api_key" \ -d '{"fingerprint_hash": "7f3d8a9c1b4e5d2f", "threshold": 0.35}' # 检查授权 curl -X POST https://portraitpayai.com/api/portrait/authorization-check \ -H "Content-Type: application/json" \ -H "X-API-Key: your_api_key" \ -d '{"face_id": 12345, "usage_type": "ai_generation", "commercial_use": true}'

Python

import requests API_KEY = "your_api_key" BASE_URL = "https://portraitpayai.com/api" headers = { "X-API-Key": API_KEY, "Content-Type": "application/json" } # 搜索肖像 search_data = { "fingerprint_hash": "7f3d8a9c1b4e5d2f", "threshold": 0.35 } result = requests.post( f"{BASE_URL}/portrait/search-by-fingerprint", json=search_data, headers=headers ).json() for match in result["matches"]: print(f"{match['name']}: {match['similarity']*100}% 相似")

JavaScript

const response = await fetch( 'https://portraitpayai.com/api/portrait/search-by-fingerprint', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': 'your_api_key' }, body: JSON.stringify({ fingerprint_hash: '7f3d8a9c1b4e5d2f', threshold: 0.35 }) } ); const data = await response.json(); console.log(`找到 ${data.matches.length} 个匹配`);
