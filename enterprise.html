授权查询 API

PortraitPay 提供标准化 API，让任何 AI 平台都可以查询肖像版权可用性并获取授权。

🔍 版权查询

查询特定肖像是否可用于特定场景

// POST /api/portrait/authorization-check { "face_id": 12345, "usage_type": "ai_generation", "commercial_use": true } // Response { "can_use": true, "authorization_status": "available", "price": 500, "license_type": "commercial", "restrictions": ["no_porn", "no_politics"] }

📝 按指纹搜索

上传图像，本地生成指纹，查询是否与注册肖像匹配

// POST /api/portrait/search-by-fingerprint { "fingerprint_hash": "7f3d8a9c...", "fingerprint_type": "dhash", "threshold": 0.85 } // Response { "matches": [ { "face_id": 12345, "name": "刘德华", "similarity": 0.92, "authorization_status": "available" } ], "total": 1 }

🔐 Local-First 架构

企业客户可以：

  • 在本地设备生成指纹，原始图像不离开客户环境
  • 服务器只存储不可逆的检索索引
  • 支持私有化部署选项
  • 满足严格的数据合规要求
