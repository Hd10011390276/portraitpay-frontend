## ⛓️ Blockchain Configuration

### PortraitCert Contract (Ethereum Sepolia Testnet)
- **Network:** Ethereum Sepolia (chainId: 11155111)
- **Contract Address:** `PORTRAIT_CERT_CONTRACT_ADDRESS` (in Vercel env)
- **ABI:** `src/lib/blockchain/abi.ts`
- **RPC:** `ETHEREUM_SEPOLIA_RPC_URL`
- **Signing:** Platform wallet (`ETH_WALLET_PRIVATE_KEY`) — NOT the user's wallet

### Licensing Contract (Base Mainnet)
- **Network:** Base Mainnet (chainId: 8453)
- **Contract Address:** `LICENSING_CONTRACT_ADDRESS` (in Vercel env)
- **ABI:** `src/lib/blockchain/abi.ts`

### Key Design Decision (MVP)
> Blockchain certification uses the **platform wallet** (`ETH_WALLET_PRIVATE_KEY`), NOT the user's wallet.
> Users do NOT need to bind their own crypto wallet to use the platform.
> `walletAddress` field in user table is for display only.

## 🚀 Current Features (MVP)

### Working
- ✅ Portrait upload with local storage (IndexedDB)
- ✅ Face matching (face-api.js, browser-based)
- ✅ ID card verification (human face vs ID photo)
- ✅ Blockchain certification (Sepolia, platform wallet)
- ✅ IPFS metadata storage (Pinata)
- ✅ Portrait listing with status filters
- ✅ Portrait soft delete
- ✅ Duplicate image detection (SHA-256 + unique constraint)
- ✅ User registration / login (NextAuth)
- ✅ KYC merged into portrait upload flow
- ✅ Responsive dark mode UI
- ✅ Multi-language (zh-CN / en)

### Known Limitations
- ⚠️ Images stored in browser IndexedDB only (not on cloud)
- ⚠️ Sepolia = testnet only, not for production money
- ⚠️ Pinata free tier: 1GB storage, 100 files
- ⚠️ Vercel Hobby: 100GB/month bandwidth
